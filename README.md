Description

This project is a simple and secure login system built using Python. It demonstrates how authentication systems protect against unauthorized access by limiting repeated incorrect password attempts. After a fixed number of failed login attempts, the user account is temporarily locked for a short duration. This helps prevent brute-force attacks and improves overall system security.
The project is useful for understanding basic authentication mechanisms and security practices in software development.
Technologies Used

Python
Time Module (for lock duration handling)
Basic Data Structures (Dictionary for user management)
